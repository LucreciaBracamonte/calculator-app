# Calculator App

A responsive calculator web application built with HTML, CSS, and JavaScript. This project performs basic arithmetic operations through a clean and intuitive user interface.

## Features

* Addition, subtraction, multiplication, and division
* Responsive design
* User-friendly interface
* Fast and lightweight

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```
calculator-app/
│── index.html
│── style.css
│── script.js
└── README.md
```
